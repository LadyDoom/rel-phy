Ce répertoire est une ressource éducative libre (REL) qui propose des simulations interactives autour du thème des référentiels en physique classique et relativiste. Trois modules sont inclus :

1. Projectile dans un référentiel en rotation
2. Expérience Michelson-Morley
3. Diagrammes de Minkowski

# Pour utiliser ces ressources

 C'est ici: [https://physiqueudem.github.io/rel_referentiels/](https://physiqueudem.github.io/rel_referentiels/)

 Chaque module inclut une **mise en contexte**, un **guide d’utilisation** et des **suggestions d’activités pour les enseignant·e·s**. Le code est ouvert et modifiable pour une intégration facile en classe ou en auto‑apprentissage.

# Pour modifier les codes et/ou contribuer au projet

 Les codes source des trois simulations sont disponibles sur le [site](https://physiqueudem.github.io/rel_referentiels/) ou [ici](https://github.com/PhysiqueUdeM/rel_referentiels/tree/master/codes). Si vous êtes familiers avec la programmation, vous pouvez copier ou télécharger ces codes et les utiliser dans la plateforme de votre choix. 

 Si vous souhaitez pouvoir éditer et lancer les simulations directement dans le navigateur, sans avoir besoin d’installer des logiciels supplémentaires, nous vous recommandons la plateforme [Glowscript](https://www.glowscript.org/), où vous pouvez également trouver les codes de ce projet ([ici](https://www.glowscript.org/#/user/dbphy/folder/RELreferentiels/)). Cette approche est particulièrement recommandée pour le personnel enseignant qui souhaiterait intégrer ces outils à des activités en classe. 

 Ces ressources étant ouvertes et évolutives, vos contributions sont les bienvenues. Vous pouvez proposer des améliorations dans ce répertoire (voir [documentation](https://github.com/PhysiqueUdeM/rel_referentiels/tree/master/doc)), ou encore nous suggérer des idées, signaler des problèmes ou recommander d’autres simulations.
